# PHPMD

VS Code extension for php, using phpmd.

## Config

Default :

	{
		"phpmd.enabled": true,
		"phpmd.validate.executablePath": "/usr/bin/phpmd",
		"phpmd.validate.rulesets": "cleancode,codesize,controversial,design,naming,unusedcode"
	}
